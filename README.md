# imbusy

## Requirements

- Python 3.9.7
- Version manager: asdf
- Virtualenv

## Development
- Clone repo and cd into it
- Create a virtual environment:
```sh
python -m venv .
```
- Activate the virtual environment:
```sh
source bin/activate
```
- Install project dependenices:
```sh
pip install -r requirements.txt
```
- Run connect:
```sh
python connect.py
```
